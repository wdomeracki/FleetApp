# FleetApp

Aplikacja konsolowa napisana w C# (.NET), uruchamiana w Visual Studio 2022.

---

## Opis

Projekt to aplikacja działająca w trybie konsolowym, umożliwiająca interakcję użytkownika poprzez menu tekstowe. Całość oparta jest o środowisko .NET.

---

## Funkcjonalność

- menu konsolowe
- obsługa wyborów użytkownika
- dodawanie danych
- usuwanie danych
- wyświetlanie danych
- prosta logika aplikacyjna w C#

---

## Jak działa aplikacja

Po uruchomieniu użytkownik widzi menu w konsoli.  
Na jego podstawie wybiera operacje, które wykonuje aplikacja, np. dodanie lub wyświetlenie danych.

---

## Uruchomienie

1. Otwórz plik `FleetApp.sln`
2. Poczekaj aż projekt się załaduje i przywróci zależności
3. Ustaw projekt startowy (jeśli trzeba)
4. Uruchom aplikację przyciskiem **F5**
